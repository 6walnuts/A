# 股票助手 · 部署教程

整个过程免费,大约 10 分钟。部署一次后,以后父母打开的永远是最新版,你改代码重新部署即可,他们什么都不用做。

## 项目文件

```
stock-helper/
├── index.html      主应用(界面 + 全部逻辑)
├── api/hq.js       行情中转(解决跨域和中文乱码)
├── manifest.json   PWA 配置(添加到主屏幕用)
├── sw.js           离线缓存
├── icon-192.png    图标
└── icon-512.png    图标
```

## 第一步:部署到 Vercel(二选一)

### 方法 A:命令行(最快,推荐)

前提:电脑装了 Node.js(https://nodejs.org 下载 LTS 版)。

```bash
cd stock-helper
npx vercel
```

首次运行会:

1. 让你登录/注册 Vercel(选 Continue with Email,输入邮箱,点邮件里的确认链接)
2. 问 "Set up and deploy?" → 回车(Yes)
3. 后面几个问题全部直接回车用默认值

完成后终端会显示一个网址。再运行一次 `npx vercel --prod` 得到正式网址,形如:

```
https://stock-helper-xxxx.vercel.app
```

以后更新代码,重新运行 `npx vercel --prod` 即可。

### 方法 B:网页操作(不装任何东西)

1. 注册 GitHub(https://github.com),新建仓库(New repository),名字随意,选 Public,点 Create
2. 在仓库页面点 "uploading an existing file",把 stock-helper 文件夹里的**全部文件连同 api 文件夹**拖进去(注意:直接拖整个文件夹进上传区,api 目录结构要保留),点 Commit
3. 注册 Vercel(https://vercel.com),选 "Continue with GitHub" 登录
4. 点 "Add New → Project",选择刚才的仓库,点 Import → Deploy
5. 一分钟后得到网址 `https://xxx.vercel.app`

以后更新:在 GitHub 网页上传覆盖文件,Vercel 会自动重新部署。

## 第二步:验证

在自己手机或电脑浏览器打开网址,应该能看到:

- 「自选」页有默认的贵州茅台、苹果、英伟达,价格几秒刷新一次
- 「大盘」页有上证、深证、创业板、道琼斯、纳斯达克、标普500
- 点任意股票能看分时图和 K 线
- 「添加」页搜索"茅台"或"AAPL"能出结果

注意:vercel.app 域名在国内部分网络环境可能访问不稳定。如果父母在国内且打不开,解决办法是在 Vercel 项目里绑定一个自己的域名(Settings → Domains,域名在阿里云/腾讯云买一个一年几十块),绑定后用自己域名访问通常就正常了。

## 第三步:教父母安装到 iPhone(一次性,可视频通话指导)

1. 把网址发到微信,让他们**复制链接**
2. 打开 **Safari**(自带浏览器,指南针图标),粘贴到顶部地址栏,打开
3. 点底部中间的**分享按钮**(方框加向上箭头)
4. 往下划,点「**添加到主屏幕**」→ 点右上角「添加」
5. 桌面上就有「股票助手」图标了,以后像普通 app 一样点开用

微信里直接点链接是在微信内置浏览器打开的,**没有**"添加到主屏幕",必须用 Safari。

## 使用说明(可以直接念给父母听)

- **自选**:自己关注的股票,点一下看走势图
- **大盘**:看今天大盘涨跌
- **新闻**:自动更新的 7×24 财经快讯(每分钟刷新,红色边条是重点新闻)
- **添加**:输入名字(茅台)、代码(600519 / AAPL)或拼音都能搜到,点「添加」
- **删除**:点进股票,拉到最下面点「从自选中删除」
- **提醒**:点进股票,设置"涨到某价"或"跌到某价",到价时 app 里会弹窗响铃
- 红色是涨,绿色是跌(美股也按这个习惯显示)

## 已知限制

- 提醒只在 app 打开时生效(iOS 对网页应用的后台推送限制较严)。想要更强的到价提醒,可以在券商 app 里同时设置
- 美股报价可能有约 15 分钟延迟(免费行情源的限制),A 股基本实时
- 本应用只用来看行情,不涉及任何交易和账户,买卖仍在券商 app 操作
